Notes & Equations

A Markov Chain describes how users transition between states — like web pages.

  • The rows of the matrix represent current pages.
  • The columns represent where a user might go next.
  • The stationary distribution shows the long-term probability of landing on each page.
  • Here, About is more likely to be the final stop.

How I calculated it:

  1. Solve: πP = π, with constraint π₀ + π₁ = 1
  2. Solve for π₀ = 6/13 ≈ 0.462 and π₁ = 7/13 ≈ 0.538
  3. So "About" has slightly higher PageRank
